# Projekt_KCK
Coins_detector algorithm, which detect coins on images. Algorithm is based on shapes, sizes and colours.
It can recognize objects with coin-like shapes and calculate similarity to base pictures of coins. 

- Python
- Scikit-image
- Matplotlib
- Numpy
- Scipy
- Pylab
